# Diwali_Sales_Analysis_Python_Project_1

## Project learning.

Project: Diwali Sales Analysis (Python, Pandas, Matplotlib, Seaborn)

- Performed data cleaning and preprocessing using NumPy and Pandas to handle missing values, inconsistent formats, and outliers.

- Conducted exploratory data analysis (EDA) to uncover insights related to customer demographics and purchasing behavior.

- Identified key customer segments based on state, occupation, gender, and age group, contributing to enhanced customer experience and targeted marketing strategies.

- Analyzed top-selling product categories and products, helping optimize inventory planning and align with customer demand trends.

- Visualized trends and patterns using Matplotlib and Seaborn, enabling clear communication of findings to stakeholders.
